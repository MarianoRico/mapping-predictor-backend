GET /annotations¶

Download all the annotations as a JSON or CSV.

On the headers of the HTTP request, it is possible to specify the format with:

Accept: text/plain
Accept: application/json
Parameters:
  • langa (str) – Language A (Usually ‘en’)
  • langb (str) – Language B (Use the shortest ISO code available)
Status Codes:
  • 400 Bad Request – Parameters has not been correctly established
POST /annotations¶

Add an annotation to the dataset. This endpoint should be used only by the service that generates annotations once a dbpedia release has been published.
